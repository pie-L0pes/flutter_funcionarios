# 📱 App de Funcionarios

## 📌 Descrição do projeto

Este projeto é um aplicativo mobile desenvolvido em Dart utilizando o Flutter que exibe uma lista de funcionários a partir de um arquivo JSON local.

O aplicativo permite:

Visualizar os dados dos funcionários
Navegar entre os funcionários (anterior/próximo)
Selecionar um funcionário através de um menu dropdown

Cada funcionário possui:

Nome
Cargo
Salário
Data de contratação
Avatar (imagem)

O objetivo do app é praticar consumo de JSON local, navegação entre dados e construção de interfaces com Flutter.

*Obs:
Este projeto faz parte da aprendizagem continua do curso Densenvolvimento de Sistemas do senai.*
